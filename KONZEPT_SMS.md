# Smart Modular Space (SMS) -- Erweitertes Konzept

---

## Vision

Ein mobiles, modulares Raumsystem in Holzbauweise, das sich als Premium-Sauna, autarkes Home Office oder vollwertiger Wohnraum flexibel an das Leben des Nutzers anpasst. Per interaktivem 3D-Konfigurator in Echtzeit zusammenstellbar -- vom einzelnen Saunakern bis zum kompletten Spa Resort.

## 5. Wand- & Oeffnungssystem

### 5.1 Aussenwand-Oeffnungen

| Oeffnungstyp | Masse (Standard) | Beschreibung |
|--------------|------------------|-------------|
| **Standardtuer** | 0,9 x 2,0 m | Normaler Eingang, Kiefernholz-Rahmen |
| **Terrassentuer** | 2,0 x 2,0 m | Doppelfluegelig, bodentief, Glas |
| **Panoramafenster** | 2,0 x 2,0 m | Grosses Sichtfenster, bodentief |
| **Seitenfenster** | 1,5 x 1,5 m | Erhoeht (Bruestung 0,3m) |
| **Kleines Fenster** | 1,0 x 1,5 m | Erhoeht (Bruestung 0,3m) |
| **Saunafenster** | 0,8 x 0,6 m | Erhoeht (Bruestung 0,8m), Sauna-spezifisch |
| **Sauna-Panorama** | 2,0 x 0,5 m | Eleganter Panoramastreifen (Bruestung 1,2m) |
| **Privatsphaerefenster** | 0,5 x 0,4 m | Sehr hoch (Bruestung 1,6m), fuer Sanitaerraeume |

### 5.2 Konfigurierbare Parameter

Jede Oeffnung ist individuell anpassbar:

- **Position**: Stufenloser Schieberegler (Links <-> Rechts), angezeigt als Prozentwert oder "Mitte"
- **Breite**: Ab 0,3m, begrenzt durch Wandbreite (15cm Rand)
- **Hoehe**: 0,3m bis 2,5m
- **Bruestungshoehe (OffsetY)**: 0 bis 2,35m
- **Scharnierseite**: Links / Rechts
- **Oeffnungsrichtung**: Nach aussen / Nach innen

### 5.3 Verglasung & Material

- 2-Fach Isolier-Verglasung
- Energiesparscheiben
- Kiefernholz-Rahmen

### 5.4 Interior-Waende (Verbundwaende)

Angrenzende Module teilen sich eine Wand. Diese "Shared Wall" kann konfiguriert werden als:

- **Offener Durchgang** -- kein Trennelement
- **Trennwand** -- geschlossene Holzwand
- **Interior-Tuer** -- 0,9 x 2,0m, Kiefernholz, mit Scharnierseite und Oeffnungsrichtung
- **Interior-Terrassentuer** -- 2,0 x 2,0m Glasdurchgang

Die Shared-Wall-Erkennung erfolgt automatisch per Zell-Grenz-Analyse. Auf geteilten Waenden wird die Aussenwand unterdrueckt und durch die konfigurierbare Interior-Wand ersetzt.

**Status:** UMGESETZT -- vollstaendig im 3D-Konfigurator mit Live-Vorschau und PDF-Export.

## 7. Pergola-System (Detailliert)

### Strukturkomponenten

| Bauteil | Masse | Material |
|---------|-------|---------|
| **Pfosten** | 80 x 80 mm | Aluminium-Profil, dunkelgrau |
| **Laengstraeger** | 80 x 120 mm | Aluminium |
| **Quertraeger** | 80 x 120 mm | Aluminium |
| **Lamellen** | 40 x 60 mm, 15 cm Abstand | Aluminium |
| **Fussplatten** | 120 x 120 mm | Metall |

### Intelligente Adjazenz-Erkennung

Die Pergola erkennt automatisch pro Seite (vorne/hinten/links/rechts), ob ein Haus-Modul angrenzt:

- **Wandanschluss:** Traeger werden buendig an die Hauswand gefuehrt, keine Pfosten auf dieser Seite
- **Mit Pfosten:** Pfosten auf allen Seiten, auch neben dem Haus
- **Freistehend:** Pergola ohne Haus-Anbindung

**Regeln:**
- Pergola darf nie breiter sein als das angrenzende Haus-Modul (sonst werden alle Pfosten auf der gesamten Kante unterdrueckt)
- Geschlossene Module ignorieren Pergolas bei der Shared-Wall-Erkennung (Waende bleiben intakt)
- Mittelpfosten nur bei breiten/tiefen Pergolas (> 3,5m Spannweite)

**Status:** UMGESETZT -- vollstaendig mit allen 3 Dachtypen und intelligentem Pfostenmanagement.

## 11. Architekturregeln (Quality Assurance)

Der Konfigurator erzwingt architektonisch sinnvolle Konfigurationen:

| Regel | Beschreibung |
|-------|-------------|
| **Sauna-Zugang** | Sauna hat NIEMALS eine Aussentuer -- Zugang nur ueber Interior-Tuer (Technik, Umkleide) |
| **Sanitaer-Privatsphaere** | In Ensembles kein Ausseneingang -- Zugang nur von innen |
| **Ensemble-Eingang** | Jedes Ensemble hat einen klar definierten Haupteingang (Technik, Umkleide oder Living) |
| **Interior-Tuer-Logik** | Interior-Tuer wird auf dem EINGANGS-Modul definiert, nicht auf dem Ziel |
| **Pergola-Breite** | Pergola darf nie breiter sein als das angrenzende Haus-Modul |
| **Shared-Wall-Erkennung** | Geschlossene Module ignorieren Pergolas -- Waende bleiben intakt |
| **Oeffnungs-Clamping** | 15cm Mindestabstand zu Wandkanten, Position automatisch begrenzt |
| **Groessen-Validierung** | Nur fuer den Modultyp erlaubte Groessen verfuegbar |
